API 서버 접속 안내

API 문서와 테스트를 위한 Swagger UI 접속 방법을 안내합니다.

Swagger UI 접속
API 문서를 확인하고 직접 테스트해볼 수 있습니다.
API 서버에 접속하려면, 다음 경로로 이동하거나 아래 버튼을 클릭하세요.
/swagger-ui/index.html
H2 Console 접속
H2 데이터베이스 콘솔에 접속하여 데이터를 직접 조회하고 관리할 수 있습니다.
H2 콘솔에 접속하려면 다음 경로로 이동하거나 아래 버튼을 클릭하세요.
/h2-console
사용 방법

1. API 서버 확인

먼저 API 서버가 정상적으로 실행 중인지 확인하세요.

2. Swagger UI 접속

브라우저에서 서버 주소 + /swagger-ui/index.html 로 접속하세요.

3. API 테스트

Swagger UI에서 제공하는 인터페이스를 통해 API를 직접 테스트할 수 있습니다.

4. DB 확인

h2에서 데이터베이스 상태를 직접 확인할 수 있습니다.

예시 URL
로컬 개발:http://localhost:8080/swagger-ui/index.html
서버 배포:https://your-server.com/swagger-ui/index.html
